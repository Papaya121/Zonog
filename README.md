# Zonog

> Проект в разработке. Текущее состояние не является финальным результатом — детали и инструкции могут меняться.

## Запуск

### Сервер

```bash
npm install
npm start
```

Сервер слушает `ws://localhost:3001/`.

### Клиент

Запустите статический клиент из папки `client`:

```bash
cd client
python3 -m http.server 5173
```

Затем откройте `http://localhost:5173` в браузере.

### Быстрая симуляция 5 игроков

Откройте клиентский URL в 5 отдельных вкладках (или окнах). Комната стартует, когда подключится 5 игроков, либо примерно через 15 секунд после первого входа.

### Проверка реконнекта

1. Дождитесь начала матча.
2. Закройте вкладку в середине матча.
3. Откройте тот же URL в течение 10 секунд.

Вы должны переподключиться к той же комнате и получить тот же `player id` (хранится в `localStorage`).
